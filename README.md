# GuardianOS 🛡️

A modular streaming operating system built by Gavin Camarillo.

GuardianOS powers custom OBS overlays, real-time Twitch alerts, and fantasy-themed interactive stream experiences.

## Features

- Real-time alert engine
- OBS Browser Source integration
- Streamer.bot integration
- Fantasy-themed overlays
- Extensible module architecture
